# Albatros

**Albatros** est une application mobile inclusive de mise en relation professionnelle conçue pour offrir un accès équitable à l'emploi. Pensée pour l'accessibilité universelle, elle met en relation les candidats (étudiants/alternants) et les entreprises en brisant les barrières discriminatoires.

## Fonctionnalités Principales

* **Design 100% Inclusif :** Contraste élevé, support des lecteurs d'écran (VoiceOver / TalkBack), taille de texte ajustable, et mode sombre/clair.
* **Analyse IA des CV & Offres :** Remplissage automatique des profils candidats et des critères d'offres d'emploi grâce à l'intelligence artificielle (OpenAI) qui extrait intelligemment les données depuis les PDF.
* **Matchmaking Profils/Offres :** Les candidats peuvent rechercher et postuler aux opportunités, tandis que les entreprises publient des offres et sourcent des talents.
* **Interface Fluide et Intuitive :** Navigation simplifiée avec un onboarding éducatif et des feedbacks immédiats sur chaque action.

## Stack Technique

* **Frontend :** React Native / Expo
* **Backend :** Node.js / Express
* **Base de données :** Supabase (PostgreSQL)
* **Intelligence Artificielle :** API OpenAI (gpt-3.5-turbo)
* **Hébergement :** Render (Backend)

## Installation & Lancement (Dev)


### 2. Lancer le Frontend
```bash
cd frontend

```

### 3. Lancer le Backend
